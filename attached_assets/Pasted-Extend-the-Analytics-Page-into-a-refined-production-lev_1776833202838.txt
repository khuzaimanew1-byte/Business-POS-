# 📈 BAR GRAPH — TOP 5 PRODUCTS

## 📦 DATA LOGIC

* Show **Top 5 products**
* Sorting depends on active mode:

  * Sales → highest quantity sold
  * Profit → highest profit

---

## 🧱 BAR DESIGN (IMPORTANT)

* Bars must be **vertical rectangles**
* Avoid rounded “chubby” look — keep edges slightly sharp or minimally rounded
* Maintain **balanced spacing**:

  * Not too wide (avoid chunky look)
  * Not too spaced out (avoid empty gaps)

Each bar must:

* Scale based on real value (no fake scaling)
* Maintain proportional accuracy

---

## ⚠️ LOW VALUE STRATEGY (OPTION A — FINAL)

Apply full visual clarity system:

### 📏 Height Behavior

* Real proportional scaling (no distortion)
* Very small values → minimal height (but still visible)

### ✨ Visual Indicator (Top Highlight)

* Add a **subtle glow / highlight at the top of the bar**
* This ensures small values are noticeable without faking size

### 🔢 VALUE LABEL (TOP)

* Display value **on top of each bar**
* Always visible (not only on hover)
* Helps users understand small vs large differences instantly

---

## 📉 ZERO VALUE HANDLING

* Show a very thin minimal bar (baseline level)
* Still render product (do not remove it)
* Clearly distinguish from low but non-zero values

---

## 🎨 COLOR SYSTEM

* Each product gets a **unique color**
* Colors must:

  * Work well on dark UI
  * Be slightly vibrant but not neon-heavy
* Maintain **color consistency**:

  * Same product = same color everywhere
* New product:

  * Auto-generate a distinct color (avoid duplicates)

---
