# 📈 LINE GRAPH — FINAL TOOLTIP + UX SYSTEM (REFINED & CLEAN)

---

## 🎯 1. TOOLTIP SIZE (COMPACT & MINIMAL)

### ❌ Problem:

* Tooltip too large → blocks graph → distracts user

---

### ✅ FINAL RULE:

* Tooltip must be **small, compact, and minimal**
* Only include essential data:

---

### 📦 TOOLTIP CONTENT:

#### 🟢 SALES MODE:

* Day (or Date)
* Quantity (items sold)

#### 💰 PROFIT MODE:

* Day (or Date)
* Profit amount

#### 🕒 DAILY MODE ONLY:

* Time (HH:MM)
* Value (quantity or profit)

---

### 🚫 STRICT:

* No extra labels
* No unnecessary text
* No large padding

✔ Tooltip should feel lightweight and fast to read

---

## 📍 2. TOOLTIP POSITIONING (SMART NON-OVERLAP SYSTEM)

### CORE RULE:

Tooltip must:

* ❌ NEVER overlap the graph line
* ❌ NEVER sit directly on a point
* ❌ NEVER block data visibility

---

### ✅ ADAPTIVE POSITIONING SYSTEM:

Instead of fixed position, use **smart zones**:

* If point is in **upper area (high values)**:
  → Tooltip appears **below the point**

* If point is in **lower area**:
  → Tooltip appears **above the point**

---

### 🧠 IMPORTANT:

* Tooltip should NOT jump randomly
* It should only reposition when necessary (edge cases)
* Movement must feel controlled, not reactive

---

## 🧠 3. TOOLTIP STABILITY (NO JUMPING ISSUE FIX)

### ❌ Problem:

* Tooltip changes position on every point → unstable UX

---

### ✅ FIX:

* Tooltip should:

  * Snap to **active data point**
  * Maintain position until user moves significantly

* Do NOT reposition tooltip for minor changes in graph height

✔ Result:

* Smooth reading
* No flicker
* No constant repositioning

---

## 📌 4. POINT + TOOLTIP CONNECTION

To maintain clarity:

* Highlight active point:

  * Slightly larger
  * Sharp contrast

* Add:

  * **Thin vertical dotted line**
  * From point upward/downward toward tooltip

✔ This creates a strong visual link

---

## 🟢 5. POINT VISIBILITY (IMPROVED)

* Default:

  * Hidden
* On hover:

  * Slightly larger than before
  * More visible (but still clean)

✔ Easier tracking without clutter

---

## 🕒 6. TIME LOGIC (STRICT FIX)

### DAILY MODE:

* Show exact time-based points
* Tooltip includes:

  * Time
  * Value

---

### WEEKLY / MONTHLY / YEARLY:

* Show aggregated daily data
* Tooltip includes:

  * Date only
  * Total value

✔ No hourly data outside daily mode

---

## 📊 7. MICRO GRID SYSTEM (PRECISION UI)

Enhance graph background:

* Add **small square grid (micro cubes)**
* Include:

  * Vertical lines
  * Horizontal lines

---

### STYLE:

* Very light opacity
* Subtle contrast
* Must not overpower graph

✔ Improves:

* Focus
* Alignment
* Precision reading

---

## 🌊 8. LINE + AREA VISUAL

* Smooth continuous line
* Area under line:

  * Light color
  * Very low opacity
  * Soft gradient

✔ Keeps visual depth without distraction

---

## ⚡ 9. INTERACTION BEHAVIOR

* Tooltip:

  * Compact
  * Stable
  * Smart positioning (above/below)
  * No overlap with line

* Movement:

  * Smooth
  * No jitter
  * No rapid repositioning

---

## 🎯 FINAL RESULT

* Tooltip becomes lightweight and fast to read
* No overlap with graph line or points
* No jumping or instability
* Clear connection to data points
* Clean micro-grid improves focus
* Daily vs aggregated logic perfectly separated
* Graph feels like a professional analytics tool
