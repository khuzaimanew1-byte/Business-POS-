# 🧠 CORE DATA PRINCIPLE (FINAL + CORRECTED)

The graph must represent **real sales activity only**, with a strict separation between:

* Raw data (all sales events stored permanently)
* Processed data (used for graph rendering)

👉 Graph should never rely on artificial or fake evenly spaced data.

---

# 📊 DAILY MODE — TIME-BASED (FINAL RULE)

In **Daily mode**, the graph works with high precision time tracking:

* Each sale event is recorded with an exact timestamp:

  * Hours
  * Minutes (mandatory)

### ✔️ DATA POINT RULE:

* Every sale generates a valid data point at its exact timestamp
* Multiple sales can exist at the same time or close intervals
* There is **no artificial limit on number of points**

---

## ⚠️ UPDATED ZERO-ACTIVITY LOGIC (IMPORTANT FIX)

If no sales occur between two time intervals:

* Do NOT leave gaps in the graph
* Instead, internally generate **implicit zero values**

### BUT IMPORTANT:

* These zero values must be **non-visible (no markers)**
* They exist only to maintain:

  * Line continuity
  * Smooth graph rendering

👉 Result:

* Continuous line always
* No visual gaps
* No fake visible points

---

# 📆 WEEKLY / MONTHLY / YEARLY MODE (UPDATED RULE)

In all non-daily modes:

* Data must be grouped by **day only**
* No time (hours/minutes) should be used

---

## ✔️ DAY-LEVEL RULE

Each day becomes one data unit:

* Value = total sales OR total profit of that day

---

## ⚠️ UPDATED MISSING DAY HANDLING

If a day has no sales:

* Do NOT create a visible data point
* BUT internally treat it as:

  * value = 0

### IMPORTANT VISUAL RULE:

* Graph line must remain continuous
* No breaks or missing segments

👉 This ensures:

* Clean visual timeline
* No broken graph structure

---

# 📈 GRAPH CONTINUITY SYSTEM (FINAL FIXED MODEL)

The graph must always behave as a continuous timeline:

* No gaps in line rendering
* No discontinuous segments
* No sudden jumps without base values

Even when:

* No activity exists → line stays at 0
