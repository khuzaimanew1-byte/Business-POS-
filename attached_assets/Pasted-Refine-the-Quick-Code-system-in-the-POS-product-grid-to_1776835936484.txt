# 🧾 QUICK CODE SYSTEM — IMPROVEMENTS

## 🔖 FORMAT RULE (IMPORTANT)

* Every Quick Code must:

  * Start with **#**
  * Be in **lowercase only**
  * Use **3–4 character segments for readability**

### Example format:

* `#wi-e`
* `#ap-j`
* `#co-l`

This ensures:

* Fast typing
* Easy memorization
* Clean visual structure

## 🎨 QUICK CODE VISIBILITY (CRITICAL FIX)

Improve readability on product images:

### REQUIRED STYLE:

* Use a **high-contrast pill/label background**
* Add slight **dark translucent background overlay OR solid dark badge**
* Ensure text is readable on ANY image

### RECOMMENDED DESIGN:

* Background: dark semi-transparent (or solid dark with slight blur)
* Text color: **bright white or soft neon accent**
* Font weight: **bold**
* Slight letter spacing for clarity

### IMPORTANT:

* Must never blend into image
* Must always remain readable from distance
* Must feel premium and consistent across all product images
