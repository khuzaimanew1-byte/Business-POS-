---

# ⏱️ REAL-TIME ANALYTICS SYSTEM (EVENT-DRIVEN)

The analytics graph must behave like a **live system connected to actual sales events**, not a continuously re-rendering UI.

---

## 🔁 UPDATE TRIGGER RULE (VERY IMPORTANT)

The graph must **ONLY update when a cart is submitted (sale confirmed)**.

### ✔️ Valid trigger:

* Product added to cart → NO graph update
* Cart updated → NO graph update
* Cart submit / sale confirmation → ✅ GRAPH UPDATE ONLY

👉 This ensures:

* No unnecessary re-renders
* No RAM waste
* Clean performance optimization

---

# 📊 DAILY MODE — REAL-TIME TIME GROWTH

In Daily mode:

* Graph time should behave like a **live timeline**
* It should only generate data up to the **current system time**

### ✔️ RULES:

* If current time is 3:00 PM:

  * Graph only shows data from start → 3:00 PM
* Future time does NOT exist in dataset

---

## 📉 NO FUTURE DATA RULE (IMPORTANT FIX)

* Do NOT generate any future timestamps
* If time is ahead of current system time:

  * Treat it as **non-existent data**
  * Or value = 0 (depending on continuity needs)

👉 Result:

* No fake future prediction
* Graph stays grounded in real time

---

# 📆 YEARLY / OTHER MODES — TIME SAFETY RULE

In yearly mode (and all aggregated modes):

* Only past and current dates are valid
* Any future date:

  * Must NOT generate a visible point
  * Must be treated as:

    * value = 0 OR ignored internally

👉 BUT:

* Timeline must remain continuous
* No structural breaks in graph

---

# 📉 CONTINUITY RULE (FINAL FIXED MODEL)

Even when data is missing:

* Graph line must remain continuous
* Use implicit zero values internally
* Do NOT create visible fake points

---

# ⚡ PERFORMANCE OPTIMIZATION (CRITICAL)

### ✔️ IMPORTANT RULE:

* Graph must NOT re-render on every small UI change
* Only update on:

  * Cart submit event (sale event)

### BENEFITS:

* Reduced RAM usage
* Faster performance
* No unnecessary computations
* Stable UI rendering
