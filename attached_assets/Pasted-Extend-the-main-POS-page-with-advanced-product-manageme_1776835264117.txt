# 🧠 1. PRODUCT CATEGORY MANAGEMENT (DESKTOP / LAPTOP)

## 📦 RIGHT CLICK PRODUCT ACTIONS

On desktop and laptop screens:

When user **right-clicks on a product card**, show a contextual menu with:

### Options:

* **Move To**

  * On hover → show list of all categories
  * Selecting a category → instantly move/import product into that category

* **Select**

  * Activates selection mode
  * Product cards show checkboxes
  * Multiple products can be selected

* **Import**

  * Opens dropdown list of categories
  * Selecting category → imports selected products into that category

* **Delete**

  * Removes product (with confirmation popup before action)

---

## 🧩 SELECTION ENVIRONMENT

When “Select” mode is active:

* Top bar changes dynamically:

  * Show **selected items count**
  * Show **cross icon to exit selection mode**
  * Show **Import action button**

### Import behavior:

* Clicking import opens category dropdown
* Selected products are added to chosen category

# 📱 MOBILE BEHAVIOR
