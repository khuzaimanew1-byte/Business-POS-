## 📌 CORE RULE

* Viewable
* Editable
* Reassignable
* Removable (optional)
* **Validated for conflicts before saving**

---

## ⌨️ DEFAULT SHORTCUT LIST (SYSTEM LEVEL)

### Navigation

* Shift + P → Add Product Page
* Shift + A → Analytics Page
* Ctrl + ` → Search toggle

### Cart System

* Shift + C → Open Cart
* Enter (Search Active) → Add top result to cart

### Product Flow

* Enter → Create Product
* Shift + Enter → Create & Add Another

### Edit Mode

* Shift + E → Toggle Edit Mode

### Exit System

* Shift + Backspace → Back / Exit Page

# 💱 2. CURRENCY SYSTEM (GLOBAL USD BASE LOGIC)

## 🌍 CORE PRINCIPLE (IMPORTANT CHANGE)

👉 **USD is the base reference currency**

All currencies convert from USD:

* 1 USD = X PKR
* 1 USD = X OMR

No direct PKR ↔ OMR conversion logic.

---

## 🔁 GLOBAL EXCHANGE SYSTEM

* All prices in system internally stored in USD
* On display:

  * Converted based on selected currency
* Conversion applies globally in ONE action (not per item)

---

## 🧮 BATCH UPDATE RULE (IMPORTANT)

If currency settings change:

* Entire system updates **instantly in one pass**
* Includes:

  * POS page
  * Cart
  * Analytics
  * Add Product page
  * Inputs everywhere

No item-by-item recalculation UI lag

---

## 📊 DECIMAL PRECISION

Options:

* 1
* 2
* 3

### RULE:

* No forced trailing zeros

Examples:

* 3.4 → 3.4
* 6.7835 → 6.784 (precision 3)

---

## 💰 GLOBAL CURRENCY DISPLAY RULE

* Currency symbol updates everywhere:

  * Product cards
  * Cart
  * Inputs
  * Analytics
  * Add Product page

---

## 🇴🇲 OMR SPECIAL RULE

If currency = OMR:

* Display format:

  * value + “(OMR)”
* Example:

  * 10.5 (OMR)
