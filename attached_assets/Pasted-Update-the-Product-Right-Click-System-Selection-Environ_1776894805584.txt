# 🖱️ PRODUCT RIGHT-CLICK SYSTEM (UPDATED RULES)

## 📦 PRODUCT RIGHT-CLICK MENU (DESKTOP ONLY)

When user right-clicks on a **product card**, show custom context menu.

### Available options:

## 🔄 MOVE TO (UPDATED)
- Hover opens category submenu
- Allows moving product to any category instantly
- Updates UI + data immediately

---

## ☑️ SELECT MODE
- Enables selection mode for bulk actions
- Product becomes selectable (checkbox state internally)

---

## 🗑️ DELETE
- Deletes single product
- Must show confirmation popup:
  - “Are you sure you want to delete this product?”

- Affects UI + database instantly
